# Блок SQL — Финальный проект по аналитике данных

## Описание
SQL-запросы для анализа клиентов и транзакций e-commerce компании за период 01.06.2015 — 01.06.2016.

## Данные
- `customers` — информация о клиентах (пол, возраст, город)
- `transactions` — транзакции за период

## Запросы
1. **Клиенты с непрерывной историей за год** — список клиентов совершавших покупки каждый месяц без пропусков + средний чек, средняя сумма в месяц, количество операций
2. **Аналитика по месяцам** — средний чек, количество операций, уникальных клиентов, доля от годового итога
3. **Гендерное соотношение M/F/NA** — доля клиентов и затрат каждого пола по месяцам
4. **Возрастные группы** — анализ по возрастным группам с шагом 10 лет, поквартально

## Инструменты
- MySQL 8.0
